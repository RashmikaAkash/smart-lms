# 🚀 Smart LMS

A modern, scalable **Learning Management System (LMS)** built with Flutter.  
Designed for schools, tuition classes, and online learning platforms with **QR-based attendance, role-based access, and payment-ready architecture**.

---

## 📱 Overview

Smart LMS is a **mobile-first SaaS education platform** that simplifies class management for teachers and students.

It supports:
- Teacher & Student role-based system
- QR-based attendance tracking
- Online class management
- Scalable architecture for future payment integration

---

## ✨ Features

### 👨‍🏫 Teacher Features
- Create and manage classes
- Mark attendance via QR system
- View student attendance reports
- Manage course content

### 👨‍🎓 Student Features
- Join classes via QR scan
- View attendance history
- Access class materials
- Track learning progress

### 🔐 Authentication
- Role-based login (Teacher / Student)
- Secure session handling (future backend integration)

### 📊 System Features
- Clean and modular Flutter architecture
- Scalable SaaS-ready structure
- Multi-device support (Mobile, Web, Desktop)

---

## 🏗️ Tech Stack

- **Frontend:** Flutter (Dart)
- **State Management:** (To be added - Provider / Riverpod)
- **Backend (Planned):** Node.js / Firebase
